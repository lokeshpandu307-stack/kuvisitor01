FEATURES INCLUDED
- Kaveri University logo and campus image
- Login system with Admin and Campus Desk roles
- Dashboard counts: total, inside, exited, pending, invalid attempts
- Visitor registration with email, phone, purpose, date and time
- Purpose categories and dynamic fields
- Duplicate detection using phone/email
- Admin/Campus Desk approval
- High-Level Meeting is Admin-only approval
- Secure QR token simulation
- Visitor pass preview
- WhatsApp message preview and wa.me send link
- Valid / Invalid / Already Entered / Completed scan states
- ID card issue and return tracking
- Entry and exit workflow
- Multi-desk support simulation
- Purpose analytics
- CSV report download
- Print/Save as PDF report
- Works offline using browser localStorage
